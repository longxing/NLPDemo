用户模块

以下描述了如何使用灵云用户API，通过使用灵云用户API可以方便的将开发者自身的用户系统绑定到灵云平台上来。 方便的使用在VPR,FRP,AFR等各项生物特征识别上来，同时还可以在多个设备上使用同一个用户进行访问，对请求进行关联。

1. 灵云用户管理

灵云用户，即开发者通过灵云平台管理自己的用户或继承自身已经存在的用户系统。同时灵云平台提供了用户组的概念，方便开发者在多个应用中管理自己的用户。 用户组在创建时有共享和独占两种方式。 开发者所有应用中的用户共享，用户组按照创建时的指定共享或独占。一个用户可以从属于多个用户组。

2. 灵云用户管理相关接口

灵云用户管理相关接口在类HciCloudUser中实现：
添加一个用户到某一个组

static int hciAddUser(java.lang.String groupId, java.lang.String userId)

创建一个用户组需要指定为同一个开发者共享还是该应用独占

static int hciCreateGroup(java.lang.String groupId, int groupType)

删除一个用户组 解除该组userid和组之间的关系，该组用户不删除

static int hciDeleteGroup(java.lang.String groupId)

获取开发者该应用所能访问的组列表

static int hciGetGrouplist(java.util.ArrayList<java.lang.String> groupList)

获取开发者该应用所能访问的组列表

static int hciGetUserlist(java.lang.String groupId, java.util.ArrayList<java.lang.String> userList)

从一个组中删除一个用户

static int hciRemoveUser(java.lang.String groupId, java.lang.String userId)

设置当前用户 开发者指定当前用户

static int hciSetCurrentUser(java.lang.String userId)
